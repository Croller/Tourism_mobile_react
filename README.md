# Tourism Portal App 
Purchase of air tickets, hotel booking, life hack, wiki, and more.

### Description

Using React Native JS + Node JS



### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
react-native run-ios
```
